## VORX APP - PRODUCTION

#Requirements
- PHP 8.0
- Laravel 9 + Inertia
- VueJS 3
- Tailwind CSS

#Installation
- Clone Repository
- Setup Databse
- `composer install`
- `composer dumpautoload`
- `php artisan migrate`
- `php artisan db:seed`
- `npm install`
- `npm run dev`
